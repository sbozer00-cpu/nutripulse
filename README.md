NutriPulse Pro is a modern, data-driven nutrition dashboard designed to track your daily food intake with precision.

## Features
* **Smart Search:** Quickly find foods from a database of 150+ items.
* **Precise Calculation:** Toggle between Grams and Units (cups, pieces).
* **Live Dashboard:** Watch your protein, carbs, and fat bars update in real-time.
* **Micronutrient Tracking:** Track Calcium, Sodium, and Sugar levels with visual indicators.
* **Persistent Data:** Your data is saved locally using LocalStorage.

## How to Run
1.  Download this repository.
2.  Ensure the `data` folder contains `foods.json`.
3.  Open `index.html` in your browser.

## Technologies
* HTML5
* CSS3
* JavaScript (Vanilla ES6)
* JSON Database
